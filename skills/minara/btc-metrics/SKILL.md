# /btc-metrics — Bitcoin market metrics

**Shortcut for `minara discover btc-metrics`**

## Usage

`/btc-metrics`

No arguments needed.

## Execution

1. Run `minara discover btc-metrics`
2. Display Bitcoin market metrics (hashrate, supply data, market indicators)

This is a **read-only** command — no confirmation needed.
