# /minara-account — View account info

**Shortcut for `minara account`**

## Usage

`/minara-account`

No arguments needed.

## Execution

1. Run `minara account`
2. Display account info (email, wallet addresses, login status)

This is a **read-only** command — no confirmation needed.

To show all wallets including internal addresses, add `--show-all`.
